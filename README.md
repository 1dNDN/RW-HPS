# RW-HPS  
第三方铁锈战争服务器  
这是一个基于Netty的服务器  
旨在作为一个高性能 高可用的服务器 为玩家提供更好的游戏体验

## 构建
1.安装JDK 11+。如果你不知道怎么做，就去查.
2.运行“gradlew jar”  
3.您的jar将位于'build/libs'目录中.  
4.运行您的Jar,体验高性能服务器 请注意 本服务端不支持Win下运行
